Resources and examples for the FrontEnd to help us. Sources are organized by each file. Most sources in each file category can also be used for other code:


========================
React Sources: 
========================

https://reactrouter.com/6.28.0/start/tutorial (Basic React tutorial). 

https://react.dev/learn/describing-the-ui (Shows basics of React. Also starting page).

https://react.dev/reference/react-dom/components (Start of React hooks, how to do imports, how to export, etc.).

https://legacy.reactjs.org/docs/forms.html (React input, select, and textarea).

https://reactrouter.com/api/hooks/useNavigate (Shows how to use useNavigate).

https://api.reactrouter.com/v7/functions/react_router.useLocation (Shows an example of how useLocation can be handled). 

https://legacy.reactjs.org/docs/handling-events.html (Shows how to use React Event handlers).

https://react.dev/learn/responding-to-events (More on event handling). 

https://react.dev/learn/conditional-rendering (Shows how to display different things based on a condition. Conditions will most likely be user selections). 

https://react.dev/learn/sharing-state-between-components (Shows parent and child components. Shows how to handle data from multiple parts). 

https://react-hook-form.com/get-started (Shows more on form handling).

https://developer.mozilla.org/en-US/docs/Web/API/Window/localStorage (Shows how to use localStorage to get stored data).

https://www.w3schools.com/react/react_useeffect.asp (Shows how to use useEffect. Useful for session timeouts).

https://github.com/sksabbirhossain/megamart-ecommerce/tree/main/src (Example GitHub website that used React. Has front and back end).

https://github.com/GeorgiaCD/REACT-Bakery (Bakery website example that uses React).

https://github.com/topics/react-website (More example websites to explore if needed).
